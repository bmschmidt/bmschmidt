Este es el repositorio principal del proyecto The Programming Historian en español: /es, donde mantenemos los archivos del sitio que está público.

Para ver tutoriales o traducciones enviadas ver: Programming Historian Submissions.

Si tienes sugerencias sobre el sitio web o el proyecto, por favor haz clic en la pestaña Issues arriba y luego en New Issue para describir tu idea. Por favor ten en cuenta que esto será público. Si quieres enviarnos un correo electrónico privado, por favor contacta a alguna de las redactoras en jefe listadas en la página del Equipo de Proyecto.

Si quieres contribuir al proyecto puedes encontrar información detallada para autores, traductores, revisores y editores en la página de contribuciones en el sitio web.

Para contribuciones técnicas al proyecto o para leer sobre nuestras políticas para sub-grupos de nuevos idiomas interesados en hacer una traducción, por favor lee nuestro Project Wiki.

Nuestra administradora técnica es Zoe LeBlanc (@ZoeLeBlanc).
